# CustomCooking

CustomCooking is a Paper plugin developed for BMC. It enhances gameplay by introducing custom cooking mechanics, handling of jade currency management, and leaderboard systems.

## Features

- **Custom Cooking Mechanics**: Add unique recipes and cooking systems.
- **Jade Currency System**: Manage jade as an in-game currency with limits, cooldowns, and transactions.
- **Leaderboards**: Track and display top players based on various metrics.
- **Modular Design**: Easily extend or modify functionality using the `Function` class system.
- **SQLite Integration**: Persistent data storage for transactions and totals.
